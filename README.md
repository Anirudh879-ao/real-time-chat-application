# Real-Time Chat Application

A full-stack MERN real-time chat application built using:

- MongoDB
- Express.js
- React.js
- Node.js
- Socket.IO
- Zustand
- JWT Authentication

## Features
- Real-time messaging
- User authentication
- Online/offline status
- Image sharing
- Responsive UI
- Theme customization

## Tech Stack
Frontend:
- React
- Tailwind CSS
- Zustand

Backend:
- Node.js
- Express.js
- MongoDB
- Socket.IO

## Installation

```bash
npm install
npm run dev
